apy

apy is a simple replacemet for stuff I use awk for. Basically I just wanted to type less.

So you can:

$ echo "zero:one:two:three" | apy -F: -N, 3 2 1 0
three,two,one,zero

Instead of:

$ echo "one:two:three:four" | awk -F: '{print $4","$3","$2","$1"}

You can also use slices (think python lists).

$ echo "zero:one:two:three" | apy -F: 1:
one two three

Lastly if you leave off the offsets to use apy will list them for you.

$ echo "zero:one:two:three" | apy -F:
[0]zero [1]one [2]two [3]three
